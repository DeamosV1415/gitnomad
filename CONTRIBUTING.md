# Contributing to GitNomad

Thanks for helping make GitNomad better! 🎒

GitNomad's app and extension are **proprietary**, so this repository doesn't hold
their source code — but it *is* where the community shapes the product. The most
valuable things you can contribute are **clear bug reports** and **thoughtful
feature ideas**.

## Reporting a bug

1. **Search first** — check [existing issues](https://github.com/DeamosV1415/gitnomad/issues)
   and the [Troubleshooting guide](docs/troubleshooting.md). Your problem may
   already be known or solved.
2. **Open a bug report** using the
   [bug template](https://github.com/DeamosV1415/gitnomad/issues/new/choose).
3. **Include the details** the template asks for — phone model, Android version,
   computer OS, VS Code version, what you did, and what happened. Screenshots help
   a lot.

The clearer the report, the faster the fix.

## Suggesting a feature

- Open a [feature request](https://github.com/DeamosV1415/gitnomad/issues/new/choose),
  or float the idea in [Discussions](https://github.com/DeamosV1415/gitnomad/discussions)
  first if you want to talk it through.
- Explain the **problem** you're trying to solve, not just the solution — it helps
  us find the best fit.

## Improving the docs

Spotted a typo, an outdated step, or something confusing in the
[documentation](docs/index.md)? Docs live in this public repo, so
**pull requests are welcome** for the `docs/` and `legal/` markdown files. For
anything larger, open an issue first so we can agree on the direction.

## Be kind

Please keep discussions respectful and constructive. We're a small project — a
little patience and good faith go a long way.

## Security issues

Please **don't** file security vulnerabilities as public issues. Email them
privately to **gitnomad.app@gmail.com** so they can be fixed before disclosure.
